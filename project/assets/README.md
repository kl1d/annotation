# Sample Assets

This folder is intentionally empty in the public repository.

Add your local study files here when working with a real project:

- `videos/` for session recordings such as `.mp4`
- `logs/` for participant log folders or `.log` files
- `surveys/` for raw survey exports and mapping CSVs

For private/local work, use `project.local/` and run:

```bash
PROJECT_DIR=project.local docker compose up --build
```
