# Sample Data Directory

The public repository does not ship real study data.

When you run the app, the backend will create canonical CSV outputs here as needed, such as:

- `sessions.csv`
- `participants.csv`
- `survey_answers.csv`
- `log_events.csv`
- `timeline_events.csv`

For private/local work, use `project.local/` to keep your real data separate from the public sample.
